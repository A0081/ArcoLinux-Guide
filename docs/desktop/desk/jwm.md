# JWM

JWM（Joe's Window Manager）是一个轻量级的窗口管理器，它被设计用于在低端硬件上运行。它的特点是速度快、资源消耗少，并且易于定制

![img](../img/7y6ttmwg3xd91.jpg)

## 安装方法

```bash
sudo pacman -Syu
sudo pacman -S jwm
```
